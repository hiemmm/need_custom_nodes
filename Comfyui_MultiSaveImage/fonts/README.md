Place custom font files here.

Supported formats:
- `.ttf`
- `.otf`
- `.ttc`

The `MeuxArtisticTextPreview` node reads this directory first, before falling back to:
- `PIP_ArtisticWords/fonts`
- system font directories

Recommended usage:
- put project-specific fonts in this folder
- restart ComfyUI after adding new fonts
